
# athlena 🏋️‍♀️  
**“wisdom in fitness – track. transform. thrive.”**

athlena is a cutting-edge fitness and calorie tracking app inspired by the wisdom and strategy of athena, the greek goddess. it enables users to take control of their nutrition and fitness goals with smart, personalized tools.

---

## features  
- **personalized calorie tracking**  
  adaptive calorie management tailored to your weight, activity level, and goals (e.g., fat loss or muscle building).

- **natural language input**  
  log workouts like "10 sets back, 3 sets lat pulldown with 12 reps," and the ai calculates burned calories.

- **meal photo analysis**  
  upload photos of meals. the app analyzes their nutritional values and logs them automatically.

- **progress tracking**  
  detailed charts and insights on calorie intake, burned calories, weight changes, and more.

- **greek-inspiration**  
  sleek and modern visuals.

---

## tech stack  
- **frontend**: next.js with tailwind css  
- **backend**: node.js with express  
- **database**: postgresql or firebase  
- **ai tools**: openai api for natural language and image analysis  
- **hosting**: vercel for deployment  

---

## getting started  
1. clone the repository:  
   ```bash
   git clone https://github.com/thats-dominik/athlena.git
   cd athlena
   ```
2. install dependencies:  
   ```bash
   npm install
   ```
3. start the development server:  
   ```bash
   npm run dev
   ```
